# Dino Game

A simple and interactive browser-based game inspired by the classic Chrome Dino game. The player controls a dinosaur that must jump over obstacles to survive and score points.

## Features
- Smooth jump mechanics for player control
- Collision detection between dinosaur and obstacles
- Real-time score tracking based on survival
- Lightweight and responsive gameplay
- Simple and clean user interface

## Tech Stack
- HTML
- CSS
- JavaScript

## Project Overview
This project was built to understand core JavaScript concepts such as DOM manipulation, event handling, and game logic. It demonstrates how basic frontend technologies can be used to create an engaging browser game.
